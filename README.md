# ✈️ Smart Airport System

Smart Airport System is an intelligent web-based application designed to monitor, manage, and optimize airport operations. It provides real-time insights into flight movements, runway usage, and congestion levels to improve efficiency and decision-making.

---

## 🚀 Features

* ✈️ Real-time Flight Tracking
* 🛬 Runway Management System
* 🚦 Congestion Detection & Visualization
* 📊 Interactive Dashboard with Live Data
* 🔄 API Integration for Flight Data
* 📱 Responsive UI (Desktop & Mobile)

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Bootstrap
* Axios
* React Hooks

### Backend

* Python (Flask / FastAPI)

### Others

* REST APIs
* JSON Data Handling

## ⚙️ Installation & Setup

### 🔹 1. Clone Repository

git clone https://github.com/MAHANTESHSKANDE/smart-airport-system.git

cd smart-airport-system

---

### 🔹 2. Install Frontend Dependencies

cd frontend
npm install

---

### 🔹 3. Run Frontend

npm start

---

### 🔹 4. Setup Backend

cd backend

pip install -r requirements.txt

---

### 🔹 5. Run Backend Server

python app.py

---
